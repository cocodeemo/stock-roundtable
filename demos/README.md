## 原理

两个报告均由 Hermes Agent 扮演 6 个投资流派角色，基于真实行情（腾讯 API）+ 财务数据（AKShare）进行两轮辩论后自动生成。HTML 由 `scripts/html_report.py` 渲染，CSS 复刻 WorkBuddy 杂志级排版。
