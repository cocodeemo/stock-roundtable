## 一、底层认知

### 1.1 交易模式分类

| 模式 | 方法 | 特点 |
|---|---|---|
| 超短（情绪流） | 看量价关系 | 纯技术面 |
| 超短（消息流） | 利好消息买入 | 拼手速，易被量化收割 |
| 价值投资（传统） | 低估值高分红，长坡厚雪 | 适合大体量资金 |
| **高景气价值投机** | 现象级事件→行业拐点 | **他的流派** |

### 1.2 A 股上涨三驱动

| 类型 | 特征 |
|---|---|
| 流动性溢价 | 资金充沛→指数普涨，持续时间最短 |
| 高股息吃息 | 熊市防御，牛市后半段被高切低带动 |
| **消息事件驱动** | 他主攻方向 |

## 二、现象级事件识别

### 2.1 定义

> 事件发生后能明显改变一个行业的景气程度，触发二级市场价值重估。一般会走出历史新高。

### 2.2 五类现象级事件

| 类型 | 例子 |
|---|---|
| 科技应用重大突破（受众要广） | 2023 ChatGPT |
| 政治经济突发事件（贸易战/制裁） | 2018 芯片制裁 |
| 自然灾害/军事冲突→上游供应萎缩 | 2024 中东 BB 机爆炸 |
| 海外成功商业模式映射国内 | AppLovin→国内 AI 广告 |
| 官方重大政策（借钱炒股级别） | 2024.9.24 互换便利+回购贷款 |

### 2.3 伪现象级事件（必须辨别）

- 某部委发声支持某行业（口号无实质）
- 某企业宣布全力进入某行业
- 板块历史新高+头部业绩超预期（利好出尽）

### 2.4 现象级事件四要素

| 要素 | 标准 |
|---|---|
| 真实性 | 交叉验证，确认不是假消息 |
| 传播性 | 出圈效应，全民讨论 |
| 规模性 | 涉及行业体量足够大，有巨大想象空间 |
| 时效性 | **第一次最有价值**，后面类似消息只是情绪扰动 |

### 2.5 现象级事件的三类划分

| 类型 | 特征 | 例子 | 爆发力 | 持续性 |
|---|---|---|---|---|
| **0→1 突破期** | 技术突破使赛道商业化成为可能 | ChatGPT、iPhone4、新能源 | 极高 | 极长 |
| **1→10 产业化** | 量变到质变，大规模商业化 | 2019 新能源产业链、2025 智驾 | 高 | 长短不一 |
| **其他重塑** | 突发事件改变行业格局 | 制裁、管制、重组 | 不确定 | 大多短期 |

> 0→1 类：短期爆发极高、影响大、持续长、回调也大，但第一波后有持续大波段机会。

### 2.6 政策事件五要素

| 要素 | 关键问题 |
|---|---|
| 落地周期 | 有无明确落地时间？ |
| 数字规模 | 资金体量多大？ |
| 持续时间 | 一次性刺激还是长期？ |
| 覆盖群体 | 规模多大？ |
| 时效性 | 天天重复喊没用，除非持续撒钱 |

## 三、选股步骤

### Step 1：确认行业机会

**全新高景气行业四条件**：
1. 有政策资金支持
2. 市场足够大，未来几年确定会被全民日常使用
3. 商业化来到临界点或初期
4. 反面案例：氢能源、超导、核聚变（距离商业化太远）

**传统行业拐点五信号**：
1. 跌得够多
2. 跌的时间够长
3. 大量同行破产倒闭
4. 未来需求稳定或有增长，可能导致供需错配
5. 出现技术突破，商业价值变大

### Step 2：确认板块/细分行业

**新兴行业** → 选最有技术壁垒、无法绕过去的那一环（最上游优先）。类比：西部大开发，先富卖铲子的。

**传统行业拐点** → **反过来选下游头部企业**。他们已在行业内证明过自己，拥有极大溢价权。

### Step 3：挑选标的

| 维度 | 标准 |
|---|---|
| **业务纯度** | 景气业务最好是主营业务，比例越高越好（重组标的除外） |
| **股价低估** | 近1年最低的 150% 以内 / 近3年最低的 200% 以内 / 历史百分位 30% 以内 |
| **细分龙头** | 结合技术和营收判断，优先龙头 |
| **辨识度** | 说到这个产品就能想到这家企业。如果 A/H 两市只有这一个标的，直接选它 |

### Step 4：风险评估

**原地踏步型**：传统行业被龙头压制、涨上去就有人减持、业绩 3 年内出不来。

**潜在暴跌型**：业绩造假前科、安全事故、被制裁、友商技术大突破。

## 四、笨韭双击

> 「最硬的情况：产业和企业基本面**同时出现拐点**。」

| | 产业拐点 | 企业拐点 |
|---|---|---|
| 信号 | 现象级事件发生 | 营收持续增长+市占率提升+亏损大幅收窄 |
| 来源 | 见现象级事件识别 | 自身降本增效/新行业拉动 |

**笨韭双击 ≠ 戴维斯双击**：戴维斯双击看 PE+EPS，笨韭双击看产业+企业拐点，全程可以忽略 PE/PB，企业大亏时买入也没问题。

**笨韭单击**（退而求其次）：只有产业拐点，企业自身基本面还没出现拐点。

## 五、剑宗气宗双模式

| | 剑宗（一波流）30% | 气宗（长期格局）70% |
|---|---|---|
| 适用事件 | 海外映射/第三类事件 | 第一类 0→1 事件 |
| 特征 | 短期无业绩支撑，容易透支估值 | 业绩持续显现，高速爆发 |
| 操作 | 快进快出，破位就走 | 格局持有，等产业彻底爆发 |
| 回调 | 不格局，缩量企稳后可重新买回 | 回调大但可格局 |

**切换规则**：
- 市场流动性好→一波流生命周期更长，可更宽容
- 买的标的出现亏损→复盘选股逻辑，可能一开始就错了
- 持续暴涨（2个月4-5倍）→即使是气宗也要考虑止盈

## 六、四维评分（每维 0-100，加权，满分 100）

| 维度 | 占比 | 核心问题 |
|---|---|---|
| 景气度 | 30% | 行业实际景气数据（订单/产能/价格），不掺公司 |
| 业务纯度 | 25% | 景气业务占营收比例 |
| 估值水位 | 30% | 距历史高低点 |
| 催化剂强度 | 15% | 三个问题：事件/时间/影响 |

### 景气度（0-100，占比 30%）

行业可验证数据，不是股价。

| 分数 | 标准 |
|---|---|
| 80-100 | 需求爆发，供不应求（订单排满/产能>90%/涨价） |
| 60-79 | 景气上行（订单+30%以上/产能>75%/价格回升） |
| 40-59 | 行业平稳，有改善迹象 |
| 20-39 | 行业下行 |
| 0-19 | 全行业衰退 |

### 业务纯度（0-100，占比 25%）

| 分数 | 标准 |
|---|---|
| 80-100 | 核心业务，营收>70% |
| 60-79 | 占比 30-50% |
| 40-59 | 占比 10-30% |
| 20-39 | <10% |
| 0-19 | 纯概念 |

### 估值水位（0-100，占比 30%）

| 分数 | 距3年高 |
|---|---|
| 80-100 | >85%（底部） |
| 60-79 | 50-85% |
| 40-59 | 30-50% |
| 20-39 | 15-30% |
| 0-19 | <15%（顶部） |

### 催化剂强度（0-100，占比 15%）

打分前必须回答：触发事件？时间节点？影响多大？

| 分数 | 标准 |
|---|---|
| 80-100 | 现象级事件 |
| 60-79 | 强催化剂，有明确事件+时间+可量化影响 |
| 40-59 | 中等，事件存在但时间/影响不确定 |
| 20-39 | 弱，大方向对但引爆点模糊 |
| 0-19 | 无催化剂 |

**公式**：`景气度×0.30 + 业务纯度×0.25 + 估值水位×0.30 + 催化剂×0.15`

| 百分制 | 操作 |
|---|---|
| 80-100 | 🔥 超配 |
| 60-79 | ✅ 标配 |
| 40-59 | ⚠️ 观察仓 |
| 20-39 | ❌ 不碰 |
| 0-19 | 🚫 远离 |

## 七、高位止盈

### 宏观维度（触发一条就跑）
- 官方连发金牌提示风险
- 增量资金见顶（储蓄搬家 30%+、融资爆量、你奶奶也入市了）
- 成交量突破 10 万亿
- 卡脖子技术全面突破并商业化（估值在天上+想象空间到顶）

### 板块维度
- 新业态渗透率突破 30%
- 出现更超景气的新赛道
- 板块核心标的走势代表市场态度

### 个股维度
| 信号 | 操作 |
|---|---|
| 单日换手 40% | 准备跑 |
| 缩量加速上涨 | 见顶信号 |
| 偏概念股（毛利低、无净利）利好落地 | 破日线就跑 |
| 一波流涨幅 3 倍以上 | 破 5 日线减仓/清仓 |
| 实控人公布减持 | 考虑清仓换同行 |
| 股东人数大幅增加 | 警惕 |
| 高位搞不景气业务定增 | 利空 |
| 笨韭 ETF 止盈法 | 过半持仓破 5 日线→减仓 |

### 主升浪纪律
> 「主升浪只有一个动作——卖，没有买。越涨越加仓 = 喜事变丧事。」

## 八、交易纪律

1. 加仓四情况：常规回调(-20%+缩量) / 假黑天鹅 / 确定性利好→直接上仓位 / 长期负面政策→止损
2. 不止损的原因：「我从不看 K 线设止损。行业景气度发生根本性改变才离场。」
3. 景气投资≠右侧买入：有时左侧预判，有时右侧追入
4. 炒股切忌完美主义：不要等水落石出再动，市场交易的是预期
5. 市场共识=无效规律：全市场都知道的规律就不再有效
6. 内功三要素：兴趣→专注→逻辑→运气
