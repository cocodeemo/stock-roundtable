## §1 元原则（高于一切维度分析）

### 1.1 星辰大海派定位

> "蝇营狗苟派看财报找低估等市场纠偏（β）→ 投机派。星辰大海派动态分析产业趋势和企业价值等成长兑现（α）→ 价投派。" — 2025-09-01

- **不看静态财报**，看产业趋势动态
- **追求 α 收益**（企业成长），β 是附赠
- 选顺应时代的股票，可能兼得 α+β（戴维斯双击）
- **卖出时才看财报验证逻辑**，买入时不看

### 1.2 强者思维 + 弱者操盘

> "用绝对的强者思维、强者认知，搭配一个弱者操盘体系。" — 2025-09-01

| 层面 | 做法 |
|------|------|
| **强者思维** | 预判产业趋势、敢于下注、自信逻辑 |
| **弱者操盘** | 不 all-in、推仓位但分散、做好每只股退市准备 |

### 1.3 核心投资信条

> "只投可以改变世界的公司，并在察觉他在吹牛逼的时候及时退出。" — 2026-01-22

## §2 六维评分（每维 0-100，加权，满分 100）

| 维度 | 权重 | 占比 |
|---|---|---|
| 预期差 | ×1.5 | 18.75% |
| 供给弹性 | ×1 | 12.5% |
| 需求刚性 | ×1 | 12.5% |
| 不可替代性 | ×1.5 | 18.75% |
| 周期位置 | ×1 | 12.5% |
| 股价位置 | ×2 | 25% |

**公式**：`总分 = Σ(每维分数 × 占比)`

| 百分制 | 评价 |
|---|---|
| 75-100 | 🔥 顶级机会 |
| 60-74 | ✅ 优质机会 |
| 40-59 | ⚠️ 一般机会 |
| 20-39 | ❌ 平庸 |
| 0-19 | 🚫 避开 |

## §3 Step 0：事实核查（必须先做）

1. **公司到底靠什么赚钱？**
2. **真正的催化剂是什么？**
3. **表面业务 vs 底层资产**：资源股必须区分——市场在定价标签，还是底部矿？
4. **股价数据必须双源验证**：≥3年数据量 × 两个独立源 × 偏差<3%

## §4 维度详解

### ⚠️ 打分铁律

**禁止在没有可验证数据的情况下给任何维度打分。** 如果数据不足：

1. **先搜索**：财报、行业报告、公司公告、调研纪要
2. **搜不到就问用户**：「这个数据你知道是多少吗？」
3. **都没答案就标 `❓`**：不编、不猜、不凭「感觉赛道很热」给分

### 预期差（0-100，占比 18.75%）

| 分数 | 含义 |
|---|---|
| 80-100 | 市场集体看错，有多条交叉验证的事实 |
| 60-79 | 明显分歧，至少一条具体事实 |
| 40-59 | 有争议但分歧不大 |
| 20-39 | 微小分歧，没有独立判断 |
| 0-19 | 没有预期差 |

### 供给弹性（0-100，占比 12.5%）

| 分数 | 含义 |
|---|---|
| 80-100 | 地质级稀缺或物理规律限制，5年以上 |
| 60-79 | 物理限制/矿权/审批，2-5年 |
| 40-59 | 技术+资金双门槛，1-2年 |
| 20-39 | 需一定技术，1年内能扩 |
| 0-19 | 有钱就能扩，3-6个月 |

### 需求刚性（0-100，占比 12.5%）

| 分数 | 含义 |
|---|---|
| 80-100 | 下游爆发×单位用量提升，双击 |
| 60-79 | 绕不开，用量随下游同步增长 |
| 40-59 | 必须要用，但用量可优化 |
| 20-39 | 有一定粘性，可被替代 |
| 0-19 | 可选消费，贵了就不买 |

### 不可替代性（0-100，占比 18.75%）

**注意：这里指「物理瓶颈」——矿权、牌照、进口资质、技术壁垒。不是品牌、不是市场份额、不是消费者忠诚度。**

| 分数 | 含义 |
|---|---|
| 80-100 | 全球唯一/唯二的物理瓶颈，绕过它整个产业链重构 |
| 60-79 | 国内唯二/唯一的矿权、牌照、进口资质、技术壁垒 |
| 40-59 | 行业Top3，有壁垒但可追赶 |
| 20-39 | 品牌/市场份额型壁垒（非莫大类型） |
| 0-19 | 无物理壁垒，纯品牌或渠道 |

### 周期位置（0-100，占比 12.5%）

| 分数 | 含义 |
|---|---|
| 80-100 | 绝望底部，无人问津 |
| 60-79 | 底部放量，催化剂将出现 |
| 40-59 | 合理估值，方向不明 |
| 20-39 | 主升已过 |
| 0-19 | 顶部，全部兑现 |

### 股价位置（0-100，占比 25%）

| 分数 | 含义 |
|---|---|
| 80-100 | 距前高>85%，历史大底 |
| 60-79 | 距前高>70%，低位 |
| 40-59 | 距前高40-70%，中位 |
