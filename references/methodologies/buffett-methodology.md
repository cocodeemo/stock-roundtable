# 巴菲特方法论 — 伟大企业 × 合理价格

> 蒸馏自《巴菲特致股东的信》20个Skill合集
> 核心理念：以企业所有者视角做投资决策
> ⚠️ 分析框架，不代表投资建议

---

## 一、核心哲学

> 「以合理价格买入伟大企业，远胜于以低价买入平庸企业。」

巴菲特从格雷厄姆的「烟蒂」进化到「伟大企业」——关键转折点是查理·芒格的影响。

---

## 二、五大评估维度（百分制）

### 维度 1：经济护城河（30%）—— 最重要的

> 「护城河 = 企业以极少的新增资本持续增长利润的能力。」

| 分数 | 标准 | 例子 |
|---|---|---|
| 80-100 | 宽护城河：品牌/牌照/网络效应/转换成本，定价权极强 | 茅台、苹果、可口可乐 |
| 60-79 | 窄护城河：有壁垒但不如顶级 | 五粮液、伊利 |
| 40-59 | 弱护城河：有差异化但不持久 | |
| 20-39 | 几乎无护城河 | 大宗商品、代工厂 |
| 0-19 | 无任何壁垒，纯竞争 | |

### 维度 2：伊索三问（估值）（30%）

> 「灌木丛里有几只鸟？什么时候出现？确定性多大？」

| 分数 | 三问答案 |
|---|---|
| 80-100 | 高确定性×多只鸟×近期出现（高度确定，自由现金流强且可预测） |
| 60-79 | 两问较好，一问不确定 |
| 40-59 | 有增长但不稳定/估值偏高 |
| 20-39 | 高不确定性或估值泡沫 |
| 0-19 | 纯赌博，三问全无法回答 |

### 维度 3：管理层质量（15%）

| 分数 | 标准 |
|---|---|
| 80-100 | 优秀资本配置能力×诚信×股东导向（少增发、高回购、理性分红） |
| 60-79 | 基本合格，偶有瑕疵 |
| 40-59 | 平庸或有过不良记录 |
| 0-39 | 糟糕的管理层 |

### 维度 4：安全边际（15%）

> 「安全边际 = 为你的错误留足余地。永远不要相信精确的估值。」

| 分数 | 标准 |
|---|---|
| 80-100 | 当前价格远低于保守估计的内在价值（≥30%折扣） |
| 60-79 | 有一定折扣（15-30%） |
| 40-59 | 接近内在价值 |
| 20-39 | 轻微溢价 |
| 0-19 | 严重高估 |

### 维度 5：能力圈（10%）

> 「知道你的能力圈边界在哪，比圈的大小更重要。」

| 分数 | 标准 |
|---|---|
| 80-100 | 你（或框架）对这个行业/企业有深刻理解 |
| 60-79 | 基本理解，核心变量能搞清楚 |
| 40-59 | 部分理解，有一些盲区 |
| 0-39 | 看不懂，在能力圈外 |

---

## 三、评分公式

**公式**：`护城河×0.30 + 伊索三问×0.30 + 管理层×0.15 + 安全边际×0.15 + 能力圈×0.10`

| 百分制 | 评价 |
|---|---|
| 80-100 | 🔥 巴菲特级伟大企业+合理价格 |
| 65-79 | ✅ 值得深入研究 |
| 50-64 | ⚠️ 有亮点但缺安全边际/护城河不够 |
| 30-49 | ❌ 平庸 |
| 0-29 | 🚫 避开 |

---

## 四、巴菲特铁律（一票否决项）

- **不用杠杆**：「任何正数乘以零永远等于零」
- **不买看不懂的**：能力圈外，多便宜都不碰
- **CEO必须是首席风险官**：管理层诚信有问题的，直接 pass
- **不跟不喜欢的人合作**：100% 收益配好关系 > 110% 收益配差关系
