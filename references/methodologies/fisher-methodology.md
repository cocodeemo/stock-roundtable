# 费雪方法论 — 成长股投资之父

> 蒸馏自《怎样选择成长股》(Common Stocks and Uncommon Profits)
> 费雪 = 巴菲特从「买便宜货」进化到「买伟大企业」的关键推手
> ⚠️ 分析框架，不代表投资建议

---

## 一、核心哲学

> 「买入那些具备卓越成长潜力的企业，当它们的成长尚未被市场充分认识的时候。」

费雪 vs 格雷厄姆：

| | 格雷厄姆 | 费雪 |
|---|---|---|
| 买什么 | 统计上便宜 | 成长潜力大 |
| 核心技能 | 读财报 | 跟人聊天（草根调研） |
| 估值 | PE/PB硬标准 | 成长空间比当前价格更重要 |
| 持有 | 涨到合理就卖 | 只要成长逻辑不变，永远不卖 |
| 典型标的 | 烟蒂股 | 摩托罗拉、德州仪器 |

---

## 二、四大评估维度（百分制）

### 维度 1：管理层质量（35%）—— 费雪最看重的

> 「在投资中，人的因素比任何其他因素都重要。」

| 分数 | 标准 |
|---|---|
| 80-100 | 管理层诚信、专注、深度足够。坦诚沟通（坏消息也敢说）。不搞股权稀释 |
| 60-79 | 基本靠谱，有小瑕疵 |
| 40-59 | 平庸，或过于沉默 |
| 0-39 | 不如实披露、爱画饼、频繁减持 |

**关键信号**：
- ✅ 好消息主动分享，坏消息也不藏着
- ✅ 研发投入持续，不做短期利润粉饰
- ❌ 频繁增发稀释股东
- ❌ 管理层大量减持套现

### 维度 2：成长空间（30%）

> 「产品的市场潜力是否足够大，能让销售额在几年内大幅增长？」

| 分数 | 标准 |
|---|---|
| 80-100 | TAM巨大+渗透率低，销售额有望数年翻数倍 |
| 60-79 | 市场大但增速平稳，有增长但不是爆发 |
| 40-59 | 市场接近饱和，增量有限 |
| 0-39 | 市场萎缩，靠提价维生 |

### 维度 3：竞争优势（20%）

> 「有没有一种特质，使得其他公司难以与之竞争？」

| 分数 | 标准 |
|---|---|
| 80-100 | 有独特优势（专利、品牌、网络效应），竞争对手难以复制 |
| 60-79 | 有优势但可以追赶 |
| 40-59 | 弱优势，容易被颠覆 |
| 0-39 | 纯竞争，无壁垒 |

### 维度 4：利润率趋势（15%）

> 「利润率是在改善还是在恶化？」

| 分数 | 标准 |
|---|---|
| 80-100 | 毛利率持续扩张，成本控制极佳，有定价权 |
| 60-79 | 利润率稳定，护城河守住 |
| 40-59 | 利润率在承压，有竞争侵蚀 |
| 0-39 | 利润率持续下滑，价格战 |

---

## 三、评分公式

**公式**：`管理层×0.35 + 成长空间×0.30 + 竞争优势×0.20 + 利润率趋势×0.15`

| 百分制 | 评价 |
|---|---|
| 80-100 | 🔥 费雪级超级成长股 |
| 65-79 | ✅ 优质成长企业 |
| 50-64 | ⚠️ 有成长但不是费雪标准 |
| 30-49 | ❌ 平庸 |
| 0-29 | 🚫 避开 |

---

## 四、费雪的三个提醒

1. **草根调研（Scuttlebutt）**：不要只看年报。去跟供应商聊、跟客户聊、跟前员工聊。信息在财报之外。
2. **不要为了短期波动卖出**：「如果你选对了企业，最好的持有期是永远。」
3. **分散不如集中**：费雪认为持有 5-10 只超级成长股就够了，多了只会稀释你的注意力。

## 五、六框架全貌

| | 买什么 | 最看重 |
|---|---|---|
| 格雷厄姆 | 便宜的 | PE/PB |

| 巴菲特 | 伟大+合理价格 | 护城河+安全边际 |
| 史诗级韭菜 | 分红的 | 商业模式+穿透回报率 |
